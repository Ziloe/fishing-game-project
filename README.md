# fishing-game-project
# Fishing Game Project 🎣

## Overview

This project is an interactive fishing game developed using Godot Engine. The game includes player-controlled fishing mechanics, shop systems, pets, and save/load functionality.

## Technologies Used

* Godot Engine
* GDScript
* Game Scene Management
* Event Logic

## Features

* Fishing mechanics
* Pet system
* Shop system
* Save/load system

## Project Structure

* scripts/
* scenes/
* assets/
* shaders/

## Documentation

Includes design documentation and flowcharts for game systems.

## How to Run

Open project.godot in Godot Engine and run the project.
